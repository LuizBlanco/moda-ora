# Moda Ora

Projeto prático desenvolvido com **HTML5** e **CSS3** durante meus estudos de front-end, com base em um exercício proposto no curso da **Serliv**.

##  Sobre o projeto
O objetivo deste projeto foi recriar um layout de página no estilo moda/editorial, colocando em prática conceitos fundamentais de estruturação com HTML e estilização com CSS.

##  Tecnologias utilizadas
- HTML5
- CSS3

##  O que foi praticado neste projeto
- Estruturação de páginas com HTML semântico
- Organização visual e estilização com CSS
- Alinhamento e espaçamento de elementos
- Construção de seções e cards
- Ajustes visuais para aproximar o layout da proposta original

## Objetivo de aprendizado
Este projeto foi desenvolvido com foco em reforçar minha base em front-end, praticando a construção de interfaces e a organização do código.

##  Observação
Este projeto foi feito como parte dos meus estudos e teve como referência um exercício proposto no curso da **Serliv**.
